# 🎲 幸运掷骰子 (Web Dice Roller)

极简纯 Web 骰子工具，支持切换 1 或 2 个骰子，带有流畅的摇色子动画与音效，并支持移动端设备摇一摇（Shake）功能。

## 核心特性

### 最新更新
- ✨ **暗黑模式**: 一键切换明亮/暗黑主题，适应不同环境
- 🎵 **增强音效**: 添加结果音效和音效开关功能
- 🎨 **改善UI**: 采用现代玻璃拟态设计和3D立体效果
- 📊 **投掷统计**: 记录总投掷次数（小字低调显示）
- 🌙 **星星背景**: 暗黑模式下有动态星星背景
- 🔊 **语音播报**: 采用谷歌普通话女声，简洁播报（如"6点"）
- 🎲 **随机算法**: 使用加密级随机数生成，确保充分随机

### 原有功能
- **纯 CSS 绘制**: 没有引用任何图片素材，利用 CSS Grid 构建经典骰子阵列
- **美观现代**: 基于 Tailwind CSS，结合玻璃拟物态(Glassmorphism)及柔和阴影，呈现优秀的立体质感
- **音效支持**: 加入了无版权音效库提供的摇骰子音频
- **摇一摇互动 (Shake Detection)**: 支持在支持加速度感应器（`devicemotion`）的设备（如 iPhone/Android）上通过摇晃设备触发掷骰（注：iOS 13+ 需要在用户点击任意交互按钮时授权）

## 如何启动与测试

因为涉及到设备传感器权限以及本地文件安全策略跨域限制，强烈建议在本地起一个 HTTP 服务器：

```bash
python3 -m http.server 8000
```

然后用手机或电脑访问局域网 IP（比如 `http://192.168.1.xxx:8000`）进行测试！

## 使用说明

1. **选择骰子数量**: 点击"1 个骰子"或"2 个骰子"按钮切换
2. **投掷骰子**: 点击"开始投掷"按钮或在支持摇一摇的设备上摇晃手机
3. **切换音效**: 点击右上角的喇叭图标开启/关闭音效
4. **切换主题**: 点击右上角的月亮/太阳图标切换暗黑/明亮模式

## 技术栈

- HTML5
- Tailwind CSS
- Vanilla JavaScript
- CSS3 Animations
- Device Motion API

## 浏览器兼容性

- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 移动端浏览器（iOS Safari、Chrome Mobile）
- 摇一摇功能需要设备支持陀螺仪/加速度感应器
