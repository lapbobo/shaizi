# 🎲 幸运掷骰子 (Web Dice Roller)
极简纯 Web 骰子工具，支持切换 1 或 2 个骰子，带有流畅的摇色子动画与音效，并支持移动端设备摇一摇（Shake）功能。

## 核心特性
- **纯 CSS 绘制**: 没有引用任何图片素材，利用 CSS Grid 构建经典骰子阵列。
- **美观现代**: 基于 Tailwind CSS，结合玻璃拟物态(Glassmorphism)及柔和阴影，呈现优秀的立体质感。
- **音效支持**: 加入了无版权音效库提供的摇骰子音频。
- **摇一摇互动 (Shake Detection)**: 支持在支持加速度感应器（`devicemotion`）的设备（如 iPhone/Android）上通过摇晃设备触发掷骰。（注：iOS 13+ 需要在用户点击任意交互按钮时授权）。

## 如何启动与测试
因为涉及到设备传感器权限以及本地文件安全策略跨域限制，强烈建议在本地起一个 HTTP 服务器：
```bash
python3 -m http.server 8000
```
然后用手机或电脑访问局域网 IP（比如 `http://192.168.1.xxx:8000`）进行测试！
